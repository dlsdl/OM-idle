## 继承3进制表示 Vue+TypeScript 程序

### 项目结构
- 创建 Vite + Vue 3 + TypeScript 项目配置
- 实现核心算法：将自然数转换为继承3进制表示
- 创建 Vue 组件：输入、转换、结果展示

### 核心功能
1. **算法实现** (`src/utils/inheritedTernary.ts`)
   - 数字转3进制表示
   - 递归处理指数直到所有数字 ≤ 3
   - 格式化输出继承3进制表示

2. **Vue组件** (`src/components/InheritedTernary.vue`)
   - 自然数输入框
   - 转换按钮
   - 结果展示（支持数学公式格式）
   - 转换步骤展示

3. **类型定义** (`src/types/index.ts`)
   - Term 接口定义

### 技术栈
- Vue 3 (Composition API)
- TypeScript
- Vite
- 纯CSS样式（无外部UI库）
